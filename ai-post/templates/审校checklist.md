## 第二遍：风格审校（降AI味）

### 删套话
- [ ] 在当今时代
- [ ] 综上所述
- [ ] 值得注意的是
- [ ] 需要强调的是
- [ ] 总而言之
- [ ] 其他套话：___

### 拆AI句式
- [ ] "不是A而是B" 是否重复出现
- [ ] "不仅…还…" 是否堆砌
- [ ] 三段对称结构是否过于工整

### 替换书面词汇
- [ ] 显著提升 → 具体数字
- [ ] 充分利用 → 用好
- [ ] 进行操作 → 直接动词
- [ ] 相关工作 → 具体说什么
- [ ] 有效提升 → 具体效果

### 口语化
- [ ] "通过XX方式进行XX" → 直接说怎么做
- [ ] 过于正式表达 → 像说话一样写

### 加入个人态度
- [ ] 中立客观处 → 加明确观点
- [ ] "可能会" → "我觉得"

## 第三遍：细节打磨

### 句子节奏
- [ ] 标记每句字数
- [ ] 找出单调区域（连续5句长度相近）
- [ ] 插入短句（5-10字）或长句（25-35字）打破单调
- [ ] 大声朗读测试节奏感

### 多巴胺密度
- [ ] 每段至少1个多巴胺点（洞察/例子/比喻/细节/微幽默）
- [ ] 无连续3段空白区域

### 微幽默
- [ ] 每200字至少1个🙂微幽默点
- [ ] 无连续500字的😐无感区域

### 段落长度
- [ ] 手机屏3-5行
- [ ] 无超过7行的段落

### 最终大声朗读
- [ ] 全文读起来流畅自然
- [ ] 像作者本人在说话（不像AI报告）
- [ ] 节奏有变化
